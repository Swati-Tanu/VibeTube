## Run Locally

### Clone this Project

```
https://github.com/Swati-Tanu/VibeTube
```

### Install npm Packages

```javascript
npm i --global
```

### Run Server

```javascript
npm run start
```

### Runs the project in the development mode

[http://localhost:3000](http://localhost:3000)

### Builds the project for production

```javascript
npm run build
```
